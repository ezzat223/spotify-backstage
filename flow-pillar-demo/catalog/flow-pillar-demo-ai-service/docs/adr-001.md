# ADR 001 — Model inference API format

Status: Proposed

## Context
We need a simple, versioned JSON API for model inference that supports synchronous requests for low-latency use-cases.

## Decision
Use a single `/predict` POST endpoint accepting a JSON body with `input` and returning `{ result: ..., modelVersion: 'v1' }`.

## Consequences
- Simple client integration
- Easy A/B testing by switching modelVersion header
